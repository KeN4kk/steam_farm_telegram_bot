# Steam Farming Bot

Telegram-бот для накрутки часов в Steam через браузер (Playwright).  
Работает 24/7 на Render.com.

## 🚀 Быстрый старт

1. **Создайте репозиторий на GitHub** и загрузите туда все файлы из этого архива.
2. **Зарегистрируйтесь на [Render.com](https://render.com)** (через GitHub).
3. **Создайте новый Web Service**:
   - Подключите свой репозиторий.
   - **Name**: любое (например, `steam-farm-bot`).
   - **Environment**: выберите **Docker**.
   - **Plan**: Free.
   - Нажмите **Create Web Service**.
4. **Добавьте переменные окружения** в разделе **Environment**:
   - `BOT_TOKEN` = токен вашего бота (получить у @BotFather)
   - `ADMIN_IDS` = ваш Telegram ID (число, можно узнать у @userinfobot)
5. Через пару минут бот запустится. Отправьте ему `/start` в Telegram.

## 🎮 Команды бота
- `/start` — главное меню.
- Добавить аккаунт Steam — только логин.
- Начать фарминг — выбор игры и аккаунта. При первом запуске потребуется пароль и код 2FA (формат: `пароль:код`).
- Остановить фарминг — завершает текущую сессию.
- Моя статистика — сколько минут накручено.

## ⚠️ Примечания
- Бот использует **реальный браузер Chromium**, установленный в контейнере.
- Сессии сохраняются (куки), поэтому повторный ввод пароля не требуется, пока сессия действительна.
- На бесплатном тарифе Render сервер "засыпает" после 15 минут бездействия, но при получении команды просыпается. Для непрерывной накрутки рекомендуется использовать платный тариф или пинговать бота.
